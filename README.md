💸 Expense Tracker

A simple and intuitive Expense Tracker application that helps users manage their daily expenses, track spending habits, and stay financially organized. This project is ideal for beginners to understand CRUD operations, data handling, and user interaction.

🚀 Features
➕ Add new income and expense transactions
📋 View a list of all transactions
❌ Delete transactions easily
💰 Automatically calculate total balance
📊 Track income vs expenses
💡 Clean and user-friendly interface
🛠️ Tech Stack
Frontend: HTML, CSS, JavaScript
